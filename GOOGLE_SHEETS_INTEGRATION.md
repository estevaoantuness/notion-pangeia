# Google Sheets Integration
Integração do Google Sheets para identificação automática de usuários no bot Pangeia.

## Features
- Leitura dinâmica de dados da planilha de colaboradores
- Identificação automática de usuários por telefone
- Suporte a múltiplos formatos de telefone
- Cache TTL de 5 minutos para performance

## Configuração
```bash
GOOGLE_SHEETS_URL=https://docs.google.com/spreadsheets/d/... 
```
